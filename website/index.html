GogoLock is a fast user switching sofware for Mac OS X.

  • Click on the application dock icon to lock the screen respectively enable fast user switching.
  • Use the global shortcut opt + command + arrow right from anywehere to lock the screen respectively enable fast user switching.
  • It's Free and Open Source. Adjust it to your needs!
