GogoLock for Mac OS X.

  • Click on the application dock icon to lock the screen respectively enable the user switching dialog.
  • Use the global shortcut opt + command + right arrow from anywehere to lock the screen respectively enable the user switching dialog.
  • It's Free and Open Source. Adjust it to your needs!
